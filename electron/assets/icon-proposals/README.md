# Plumy Icon Proposals

These SVGs are concept directions based on the product's actual shape in code:

- timeline + kanban as equal primary modes
- local-first desktop app
- humans + agentic assignees
- structured planning rather than generic task tracking

Files:

- `plumy-icon-proposal-01-plume-plan.svg`
- `plumy-icon-proposal-02-schedule-plume.svg`
- `plumy-icon-proposal-03-local-orbit.svg`

All three are intentionally simple enough to be tested at small sizes for dock/taskbar/tray use.

This set was revised to be more literal:

- the plume shape is clearly feather-like
- planning bars are explicit rather than implied
- the mark avoids unrelated orbit/radar language
